# 前端轮询优化

> 任务开放 ≠ 活跃执行

---

## 核心原则

**Task 开放状态不能被误认为活跃执行。**

```typescript
const hasActiveRun = latestRun?.status === "queued" || latestRun?.status === "running";
```

只有 `hasActiveRun` 时才继续刷新 run 状态。Task `running` 只表示任务仍开放，等待用户下一条消息。

---

## 前端展示策略

### 状态 Badge

```typescript
// Task 状态
type TaskBadge = "Open" | "Completed" | "Failed";

// Run 状态
type RunBadge = "Queued" | "Running" | "Completed" | "Failed";
```

### 界面区域

| 区域 | 数据源 | 更新策略 |
|---|---|---|
| **对话区** | messages snapshot + 当前 run stream | SSE 实时 / 轮询 |
| **产物列表（右侧）** | artifacts store | run 完成后更新 |
| **文件树** | 懒加载 API | 用户展开时加载 |

## 轮询逻辑

### ✅ 正确实现

```typescript
function shouldPollRunStatus(task: Task, latestRun?: TaskRun): boolean {
  if (!latestRun) {
    return false; // 没有 run，不轮询
  }
  
  return latestRun.status === 'queued' || latestRun.status === 'running';
}

// 使用示例
if (shouldPollRunStatus(task, latestRun)) {
  subscribeToRunEvents(task.id, latestRun.id);
} else {
  // 停止轮询，仅显示历史状态
  stopPolling();
}
```

### ❌ 错误实现

```typescript
// 错误：Task status = 'running' 就一直轮询
if (task.status === 'running') {
  setInterval(() => pollTaskStatus(), 1000); // 浪费资源！
}
```

## 状态转换图

```
Task 创建
  ↓
Task: running, No active run
  ↓ 用户发送消息
Task: running, Run: queued
  ↓ Worker 认领
Task: running, Run: running ← 【开始轮询】
  ↓ Run 完成
Task: running, Run: completed ← 【停止轮询】
  ↓ 等待用户下一条消息
Task: running, No active run ← 【不轮询】
  ↓ 用户关闭任务
Task: completed ← 【不轮询】
```

## 实现示例

### React Hook

```typescript
function useTaskPolling(taskId: string) {
  const [task, setTask] = useState<Task | null>(null);
  const [latestRun, setLatestRun] = useState<TaskRun | null>(null);
  const [isPolling, setIsPolling] = useState(false);
  
  useEffect(() => {
    // 加载初始状态
    loadTaskDetail(taskId).then(detail => {
      setTask(detail.task);
      setLatestRun(detail.latestRun);
    });
  }, [taskId]);
  
  useEffect(() => {
    // 判断是否需要轮询
    const shouldPoll = latestRun?.status === 'queued' || latestRun?.status === 'running';
    
    if (shouldPoll && !isPolling) {
      setIsPolling(true);
      subscribeToRunEvents(taskId, latestRun.id);
    } else if (!shouldPoll && isPolling) {
      setIsPolling(false);
      unsubscribeFromRunEvents();
    }
    
    return () => {
      if (isPolling) {
        unsubscribeFromRunEvents();
      }
    };
  }, [taskId, latestRun, isPolling]);
  
  return { task, latestRun, isPolling };
}
```

### Fetch Streaming 订阅

**🚨 协议说明**：使用 fetch streaming，不是 EventSource

```typescript
class RunEventStream {
  private reader: ReadableStreamDefaultReader | null = null;
  private abortController: AbortController | null = null;
  
  async subscribe(taskId: string, prompt: string, onEvent: (event: TaskEventEnvelope) => void) {
    this.unsubscribe();
    
    this.abortController = new AbortController();
    
    const response = await fetch(`/api/task/${taskId}/message`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ prompt }),
      credentials: 'include',
      signal: this.abortController.signal,
    });
    
    this.reader = response.body!.getReader();
    const decoder = new TextDecoder();
    let buffer = '';
    
    while (true) {
      const { done, value } = await this.reader.read();
      if (done) break;
      
      buffer += decoder.decode(value, { stream: true });
      const lines = buffer.split('\n\n');
      buffer = lines.pop() || '';
      
      for (const line of lines) {
        if (line.startsWith('data: ')) {
          const envelope: TaskEventEnvelope = JSON.parse(line.slice(6));
          onEvent(envelope);
          
          if (envelope.type === 'turn_complete') {
            this.unsubscribe();
            return;
          }
        }
      }
    }
  }
  
  unsubscribe() {
    if (this.abortController) {
      this.abortController.abort();
      this.abortController = null;
    }
    if (this.reader) {
      this.reader.cancel();
      this.reader = null;
    }
  }
}
```

## 性能优化

### 避免不必要的请求

```typescript
// 缓存 Task 详情，仅在状态变化时刷新
const taskCache = new Map<string, { task: Task; timestamp: number }>();

async function getTaskWithCache(taskId: string, maxAge: number = 5000): Promise<Task> {
  const cached = taskCache.get(taskId);
  
  if (cached && Date.now() - cached.timestamp < maxAge) {
    return cached.task;
  }
  
  const task = await fetchTaskDetail(taskId);
  taskCache.set(taskId, { task, timestamp: Date.now() });
  
  return task;
}
```

### 批量更新 UI

```typescript
// 使用 requestAnimationFrame 批量更新
class EventBatcher {
  private pendingEvents: TaskEventEnvelope[] = [];
  private rafId: number | null = null;
  
  addEvent(event: TaskEventEnvelope) {
    this.pendingEvents.push(event);
    
    if (!this.rafId) {
      this.rafId = requestAnimationFrame(() => {
        this.flush();
      });
    }
  }
  
  private flush() {
    const events = this.pendingEvents.slice();
    this.pendingEvents = [];
    this.rafId = null;
    
    // 一次性更新 UI
    updateUI(events);
  }
}
```
